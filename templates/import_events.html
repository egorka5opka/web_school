Импорт событий


Для импорта событий из файла:

  1. Создайте текстовый файл
  2. Для каждого события поворите шаги 3 - 5
  3. В одно строку разделив точкой с запятой запишите название и год события (обязательно число)
    Пример: "Создание интернета;1983"
  4. Далее текстом запишите описание события
  5. После конца описания поставьте сочетание символов "*/" (без кавычек)
  6. Сохраните файл и загрузите ниже
